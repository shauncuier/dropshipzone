# Security Policy

## Supported Versions

Specifically, we only provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take security seriously. If you discover any security-related issues, please avoid using the public issue tracker. Instead, email us directly at support@dropshipzone.com.au.

Please include the following in your report:

-   A description of the vulnerability.
-   The potential impact of the vulnerability.
-   Steps to reproduce the vulnerability.
-   Any potential fixes or mitigations you're aware of.

We will acknowledge your report within 48 hours and provide a timeline for a fix within 5 business days. We will keep you informed of our progress throughout the process.

Thank you for helping us keep the Dropshipzone community safe!
